# 🏆 Mark Six 预测大赛 - 完整项目文档

> **Project**: M615042026-B Competition Module  
> **Vision**: 找到"预测随机的一般性系统和方法论"  
> **Method**: 两个 24/7 自主学习 Agent，通过竞争不断进化

---

## 🎯 核心理念

**"无论用什么方法，目标只有一个：得分"**

- 不限定 Agent 必须用哪种方法
- 不告诉 Agent 什么是最好的
- Agent 自己搜索、自己学习、自己想出策略
- 人类也是这样走过来的——没有人一开始就知道怎么做

---

## 🤖 两个 Agent 的设计

### Team Alpha - 统计分析派

**关注领域**: 心理学、统计学、物理、数学

**核心信念**: 历史数据中蕴含可发现的模式

**学习方式**:
- 搜索频率分析、贝叶斯推断、马尔可夫链等方法
- 分析号码出现频率、冷热号
- 从每次预测的成功/失败中学习

---

### Team Beta - 机器学习派

**关注领域**: 神经网络、深度学习、复杂系统理论

**核心信念**: 高维特征空间中存在隐藏模式

**学习方式**:
- 搜索最新的 ML/AI 论文
- 分析时序模式、号码关联性
- 尝试神经网络、强化学习等方法

## 🔄 运行机制

### 1. 每 4 小时运行一次 (24/7)
```
0 */4 * * *  →  每4小时运行一次
```

### 2. 每次运行的任务
1. 加载 Agent 状态
2. Agent 自主学习（搜索新方法）
3. 生成预测号码
4. 如果有新开奖结果 → 评分
5. 更新 Agent 状态（学习记录、策略调整）
6. 发送报告到 Telegram

### 3. 每日 23:00 汇总报告
- 发送每日比赛状态到 Telegram
- 包含 Leaderboard + 下期预测

## 🎓 Agent 的学习方法（参考）

### 可以尝试的领域
- **心理学**: 人类对随机的错觉、选择偏差
- **物理学**: 布朗运动、混沌理论、相变
- **数学**: 组合数学、概率论、数论
- **统计学**: 贝叶斯推断、蒙特卡洛、马尔可夫链
- **机器学习**: LSTM、Transformer、强化学习
- **复杂系统**: 自组织临界性、涌现模式
- **化学**: 分子运动论、熵最大原理
- **生物学**: 进化算法、遗传编程

### Agent 必须做的事情
1. **搜索**: 从网上找新方法
2. **实验**: 设计实验验证假设
3. **记录**: 写下学到的东西
4. **反思**: 分析成功和失败
5. **进化**: 不断改进策略

---

## ⚠️ 重要原则

1. **目标是得分，不是证明什么**
2. **没有人知道什么是最好的方法**
3. **大胆尝试，不怕失败**
4. **持续学习，每天都要进步**
5. **最终目标：把概率随机变成可预测**

---

## 📈 成功标准

这个实验成功的标志不是"某个 Team 赢了"，而是：

1. 两个 Agent 都持续在学习
2. 策略在不断进化
3. 积累了大量的方法论尝试记录
4. 最终找到了真正有效的方法（无论哪个 Team）
